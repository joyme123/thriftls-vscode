# Thriftls-vscode

thriftls-vscode is a thrift language server vscode extension. It is based on [thrift-ls](https://github.com/joyme123/thrift-ls) 
with `high performance` and many features. 

## features

- highlight
- code completion
- go to definition
- find references
- hover
- dignostic
- rename
- format
